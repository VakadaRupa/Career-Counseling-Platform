# CareerCounselingPlatform_Career (Frontend)

## 📌 Project Description
CareerCounselingPlatform_Career is a web-based career counseling platform designed to help users explore career paths, book counseling sessions, access learning resources, and connect with career advisors.

The frontend is built using React, Tailwind CSS, and ShadCN UI components for a modern and responsive user experience.

---

## ✨ Features
- User authentication (Login/Register)
- Profile creation and management
- Counselor browsing and matching
- Session booking and scheduling
- Resource library (Articles, Videos, Templates)
- Job board integration
- Community forum
- Career assessment tools
- Career path visualization
- Progress tracking dashboard
- Dark mode support
- Mobile responsive design

---

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- ShadCN UI
- Axios
- Context API (State Management)
- React Router
- Chart Libraries (if used)
